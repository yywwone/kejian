常见问题
1.什么是欧朋浏览器？

Opera欧朋浏览器是一款适用于各种手机平台的手机浏览器，体积小巧且功能强大。界面设计风格简约，操控方便。速度快、超省流量。您在刷微博、看小说、读新闻的时候，都能享受到舒适的浏览体验。
2. 欧朋浏览器的优势有哪些？

节约流量，数据通过服务器压缩，可为您节约最多 90% 的流量。 （可通过选择图片压缩质量等进行设置），与同类产品相比优势比较明显。体积小，适应性好，同时支持智能/非智能手机。通用版本是 Java 程序，只要您的手机支持 Java 那就可以使用欧朋手机浏览器全部功能。能够在国内市场上绝大部分支持 Java 的手机上运行。同时欧朋手机浏览器也针对 Symbian 和 Android 平台发布了性能更好的专版。内容丰富，欧朋手机浏览器不但可以浏览 Wap 页面，更可以访问 Web 页面，支持缩放浏览（类似与 iPhone 浏览方式）您可以在手机上方便的浏览以前只有在电脑上才能看的网站。
3. 欧朋的安全性如何？

欧朋手机浏览器是目前安全性最好的手机浏览器。欧朋手机浏览器传输数据，包括从手机端到服务器端都是加密传输，无法通过拦截数据来得到您的信息。
4. 我可以在哪里找到欧朋浏览器的版本号？

在帮助 > 关于欧朋中可以看到 “Version x.x.xxxx” 的字样，这串数字即为欧朋手机浏览器的版本号。

7. 欧朋浏览器支持哪些手机平台？

欧朋手机浏览器目前发布了 Java、Symbian、Android 版本，iOS 及其他平台版本正在开发中。
8. 欧朋浏览器支持 Windows Mobile 手机吗？

运行欧朋手机浏览器需要 Windows Mobile 上安装 Java 虚拟机。某些 Windows Mobile 手机版本缺省不带 Java 虚拟机，或有问题。只需重新在 Windows Mobile上安装 Java 虚拟机，根据测试安装 jbed，或者 jblend 虚拟机均可正常使用欧朋手机浏览器。
9. 欧朋浏览器可以运行于其他非手机设备上吗？

理论上讲只要该设备能够运行 Java 虚拟机，那就可以使用欧朋手机浏览器，比如 PSP。 但需要注意相应的网络设置是否正确。一般来讲如果该设备上其他的 Java 程序可以访问网络，则欧朋浏览器亦可正常使用。
10. 欧朋浏览器多久更新一次？我该如何升级？

因为欧朋手机浏览器是服务器+客户端模式，因此我们的很多更新是在服务器端完成的，您不需要做任何更新。 如果有客户端更新，我们会欧朋手机浏览器首页上显示更新提示，您可以根据提示下载安装更新即可。 我们的官方博客即将开通，届时您亦可关注我们的官方博客了解更多更新详情。
11. 欧朋浏览器安装时为什么会白屏？

欧朋手机浏览器安装时需要连接服务器下载相应手机配置，所以在网络连接不好或者网速较慢时会出现白屏，如果您的手机支持 WIFI，建议您使用 WIFI 网络安装欧朋手机浏览器。
12. 为什么点击下载后没有收到短信？

如果手机装了360手机卫士，就无法收到短信提醒，在360的骚扰拦截里找到信息，点击链接下载即可。
13. 欧朋浏览器可以使用 cmwap (移动梦网)？

可以，欧朋手机浏览器可以使用 cmnet 和 cmwap 两种接入点上网。安装时程序会自动检测。高级用户注意，如果用 cmwap 上网，连接设置必须选为 http, 而不是 socket。cmnet 则两种连接都可以选用。
14. 欧朋手机浏览器是否支持多标签浏览？

支持，键盘机将焦点移至链接上，按 “1” 键选择 “在新标签打开” 即可。 触屏机长按链接选择 “在新标签打开”即可。
15. 欧朋手机浏览器字体太大，如何调整？

您可以在工具→设置中设置字体大小，目前仅支持设置大、中、小三种字号。对于 Java 版本，最小字号受限于您手机的 Java 虚拟机
16. 如何添加网站到首页快速拨号？

点击首页快速拨号最末尾的加号按钮，您可以从列表中直接选择您喜欢的站点，或者在顶部地址栏中填入您要添加的站点 URL 即可。
17. 如何删除一个快速拨号？

键盘机将焦点移至要删除的快速拨号图标上，按“1”键，选择清除即可。 触屏机长按快速拨号图标，选择清除即可。
18. 欧朋手机浏览器是否支持直接下载文件？

根据手机配置，欧朋手机浏览器支持直接下载文件，或调用手机本地浏览器进行下载。 如果您手机的 Java 虚拟机支持 JSR-75 标准，那么欧朋手机浏览器可以直接进行下载。市面上大部分诺基亚，索爱等手机均支持该种下载。 注意，某些支持数字签名的手机，比如诺基亚，索爱等可能需要安装签名版。 如果您的手机不支持 JSR-75标准，欧朋手机浏览器会调用手机上本地浏览器进行下载。欧朋手机浏览器的下载管理功能正在开发中，将近期将推出。
19. 欧朋手机浏览器是否支持书签同步？

欧朋手机浏览器自 6.2 版起加入了书签同步功能，在设置中启用 Opera Link 即可。
20. 如何最快速的添加自定义搜索？

这在欧朋手机浏览器中非常容易。 使用欧朋手机浏览器访问您要添加的搜索引擎，如 http://www.baidu.com，将焦点移至搜索框上，键盘机按“1”键选择“添加搜索”即可；触屏手机长按搜索框选择“添加搜索”即可。
21. 为什么我无法访问某些 wap 站点？

某些 wap 站点出于各种考虑限定只有移动/联通网内 IP 才可访问，而欧朋手机浏览器是通过服务器中转的，可能被阻止。 网站管理员只需取消对欧朋手机浏览器服务器的阻止即可。我们会积极同这些网站联系。 同时也希望用户直接向这些网站客服部门反应，有助于他们放开对欧朋手机浏览器服务器的限制。
22. 欧朋手机浏览器是否支持 Flash 动画？

下载 Flash 动画本身可能耗用大量带宽，大部分手机目前的性能也不足以支持 Flash 的流畅播放，所以欧朋手机浏览器目前并不支持播放 Flash 动画。
